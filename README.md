# Job-Tracker (IN-progress...)

#### Instead of using Excel to log my job hunt.. I created my own application with a MERN Stack! 

## Technologies Used:

* MongoDB
* Express
* React.js
* Node



## Features:
* Create a job with the company name, title, date, & response
* Edit, update, & delete functionality
